分战利品

2026/4/18 test

题目背景

小鸟游星野和ABYDOS的几位小伙伴们打死了一个总力战BOSS，她们非常高兴。想到有些学校还因为这些战利品面临着关停的风险，于是星野果断把这些战利品捐赠给了SCHALE，由身为sensei的你统一分发。

题目描述

现在来分战利品的有 $n$ 个代表，保证 $n \ge 2$。

但由于你只是个平平无奇的正常人，所以你的体力有限，至多只能从星野那里拿 $R$ 个战利品回去。

但是拿的太少不够分的，所以你至少要拿 $L$ 个战利品回去。保证 $n \le L \le R$。

也就是说，如果你拿了 $k$ 个战利品，那么你需要保证 $L \le k \le R$。

如果你拿了 $k$ 个战利品，你将把这 $k$ 个战利品放到什庭之匣里，并要求大家按照如下方案分发战利品：只要什庭之匣里有不少于 $n$ 个战利品，学院代表的所有 $n$ 个人（包括你自己）都从什庭之匣中拿走恰好一个战利品，直到什庭之匣里的战利品数量少于 $n$ 个。此时什庭之匣里剩余的战利品均归你所有——这些战利品作为你的中介费。

作为有师德的sensei，你希望让作为你的中介费的战利品数量尽可能多；因此你需要写一个程序，依次输入 $n, L, R$，并输出你最多能获得多少作为你中介费的战利品数量。

输入格式

输入一行，包含三个正整数 $n, L, R$，分别表示学院代表的个数、战利品数量的下界和上界。

输出格式

输出一行一个整数，表示你最多能获得的作为你的中介费的战利品数量。

输入输出样例 #1

输入 #1

7 16 23

输出 #1

6

输入输出样例 #2

输入 #2

10 14 18

输出 #2

8

说明/提示

【C#注意】

如样例所示，若想在C#中输入三个用空格隔开的数，则需要：

string[] inputs = Console.ReadLine().Split(' '); // 用空格分割输入
int a = int.Parse(inputs[0]);
int b = int.Parse(inputs[1]);
int c = int.Parse(inputs[2]);

对于上述代码，输入

7 16 23

后，变量值分别为

a=7
b=16
c=23
